
# AI-Powered Smart Recycling Bin (AIoT) ♻️💻

An end-to-end Smart Waste Management prototype that automates waste sorting in real-time. It leverages Computer Vision for classification and IoT for cloud analytics.

## 🛠️ Tech Stack
- AI/ML: YOLOv8 (Ultralytics), OpenCV
- Microcontrollers: ESP32, Servo Motor, OLED Display (SSD1306)
- IoT Platform: Blynk Cloud Dashboard
- Language: Python, C++ (Arduino)

## 🚀 Features
1. Real-time Classification: Detects and classifies waste items as Plastic or Metal via camera stream.
2. Automated Sorting: Routes the classified waste to its designated compartment using a servo-actuated lid.
3. Local UI: Shows system status and classifications locally on an OLED display.
4. Cloud Analytics: Real-time synchronization of waste counters and bin levels on Blynk Cloud.
